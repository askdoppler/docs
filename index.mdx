## Integration

We provide an easy-to-plug SDK so you can get started within minutes. To do so create an account on the platform, generate an API Key and follow installation tutorials for one of our compatible frameworks.

### API Key

Please generate an API Key under [Doppler](https://askdoppler.com) > [Secrets](https://askdoppler.com/secrets) > "Create new API Key" button
